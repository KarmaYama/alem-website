Alem CLI

The Alem CLI is a command line interface (CLI) for the Alem Platform. It's a simple and intuitive way to interact with the Alem Platform. It's designed to be easy to use and easy to learn.

Install Alem CLI

Requirements

  • Windows 10/11, Mac OSX, Linux
  • Node.js v12 or higher
  • NPM

Install

Mac (homebrew)

$ brew install alem-cli

NPM

$ npm install alem-cli

Yarn

$ yarn install alem-cli
